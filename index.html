This Website can solve/generate random matchstick problems/puzzels.
Comments and feedback are welcome on the Github Project Page
You can enter your puzzle into the input box below (for example: 6+4=4) and the program will generate ALL possible solutions with detailed explentation (by moving one matchstick) and will even generate images of the problem (that you can share) and the solutions.
This algorithm works by looking up a hard coded map of all numbers/symbols and what they can turn into, for example 5:

If one matchstick is taken, 5 can become 0 valid numbers
If one matchstick is given, 5 can become: 6 and 9
If one matchstick is moved within 5, 5 can become: 3
